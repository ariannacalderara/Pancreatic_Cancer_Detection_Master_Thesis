# Pancreatic Cancer Detection: Master Thesis

This repository contains the code, data processing pipelines, and experimental results developed for the Master Thesis titled Towards Accountable AI for PDAC: A Multimodal XAI Evaluation. 

This project explores the application  and quantitative evaluation of Explainable AI (SHAP and LIME) to enhance the automated detection of pancreatic cancer, aiming to improvediagnostic accuracy.


## Project Overview
Pancreatic cancer is often diagnosed at late stages due to the difficulty of early detection. This research project focuses on tabular urinary biomarker data to identify malignant features. The model leverages black box classification models (RF, MLP, SVM) and Explainable AI techniques like SHAP and LIME to achieve early stage diagnosis.
